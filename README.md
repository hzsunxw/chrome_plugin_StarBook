## 📖 项目简介

**Smart Bookmarker** 是一款革命性的Chrome浏览器扩展，利用人工智能技术重新定义书签管理体验。通过先进的AI算法，自动分析、分类和总结您的网页书签，让信息整理变得智能高效。

> 🚀 **业界首个集成多AI提供商的书签管理工具** | 🌟 **智能新标签页，打造个人工作门户** | 💫 **完全开源，社区驱动发展**

## ✨ 核心特色

### 🤖 AI智能分析引擎
- **自动内容摘要**: 智能提取网页核心内容，生成精准摘要
- **智能标签生成**: AI自动分析内容并生成相关标签
- **多维度分类**: 基于主题、内容类型、阅读难度等多维度智能分类
- **置信度评分**: 每个分类都附带AI置信度评分，确保准确性

### 🔍 智能搜索与问答
- **自然语言搜索**: 使用自然语言查询书签内容
- **语义理解**: 基于内容语义的智能推荐系统
- **问答系统**: 针对书签内容的交互式问答功能
- **学习助手**: 内置学习工具，帮助巩固知识记忆

### 🌐 多平台同步
- **实时同步**: WebSocket实现跨设备实时同步
- **云端备份**: 可选云存储，数据安全无忧

## 🎉 快速开始

### Chrome扩展安装
```bash
# 克隆项目
git clone https://github.com/your-repo/smart-bookmarker.git

# 加载扩展
1. 打开 Chrome 浏览器
2. 访问 chrome://extensions/
3. 启用开发者模式
4. 点击"加载已解压的扩展程序"
5. 选择项目目录
```

### 服务器部署（可选）
```bash
cd server
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，配置数据库和AI密钥

# 启动开发服务器
npm run dev

# 或启动生产服务器
npm start
```

### 配置AI提供商
1. 打开扩展设置页面
2. 选择您偏好的AI提供商（OpenAI、DeepSeek、OpenRouter）
3. 输入相应的API密钥
4. 配置分析深度和同步选项
